## Getting Started
```sh
$ npm install 
```

## Available Scripts

In the project directory, you can run:

### `$ npm start`

Runs the app only view mode.
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.


### `$ npm test`

Launches simple jest test to verify celsius converter.
